# Mini Compiler Project

This project implements a simple Mini Compiler in Python.

## Features
- Lexical Analysis
- Syntax Analysis
- Semantic Analysis
- Intermediate Code Generation
- Code Optimization
- Target Code Generation

## Requirements
- Python 3.x

## How to Run
1. Open terminal or command prompt
2. Navigate to project folder
3. Run the following command:

   python compiler.py

## Sample Input
int a = 5 + 3;

## Sample Output
Displays tokens, intermediate code, optimized code, and target code.
